# SIMPLE CALCULATOR (Python)

## Project Description
This is a simple command-line calculator built using Python.  
It allows the user to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.  
The user selects an operation from a menu and inputs two numbers. The program then displays the result.

## Features
- Menu-driven interface
- Supports addition, subtraction, multiplication, and division
- Input validation for numbers
- Handles division by zero with a proper error message
- Runs in any terminal/command prompt with Python installed
